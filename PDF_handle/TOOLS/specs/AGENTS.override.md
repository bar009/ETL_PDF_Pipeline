# AGENTS.override.md

Local override for `PDF_handle/TOOLS/specs/`.

## Purpose

This folder contains operational specs, architecture notes, and phase contracts for the tool layer.

## Rules

1. Treat these files as operator guidance and design references, not executable truth by themselves.
2. Keep spec names stable once other docs or tooling link to them.
3. When behavior changes, update the spec closest to the affected tool or phase.
4. Prefer cross-linking related specs here instead of duplicating long policy text in many places.
