# Before/After Heuristics

Use this file when the three modes are drifting back toward one generic UI.

## Learning

### Before

- homepage behaves like a catalog
- topic page behaves like a record
- “next” logic is buried
- provenance and metadata compete with the lesson

### After

- homepage gives a guided start
- topic page starts with lesson value
- the next step is obvious
- advanced material sits lower

## Encyclopedia

### Before

- homepage pushes a path instead of a map
- search is available but not central
- topic page feels too tutorial-like

### After

- homepage feels like a knowledge atlas
- search and category browsing are first-class
- topic page answers “what is this and where does it belong”

## Research

### Before

- homepage feels like a generic browse screen
- topic page buries evidence under summary
- comparison entry points are weak

### After

- homepage surfaces source-backed or relation-heavy entries early
- topic page shows claim plus support quickly
- comparison and provenance are easy to reach

## Shared Smell

If all three modes still share the same:

- hero rhythm
- section order
- metadata emphasis
- related-link prominence

then the mode system is still under-designed.
